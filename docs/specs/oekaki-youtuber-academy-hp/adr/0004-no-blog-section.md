# ADR-0004: ブログセクションを当面実装しない

| 項目 | 値 |
|---|---|
| 状態 | Accepted |
| 日付 | 2026-06-28 |
| 関連要件 | requirements §2.2, FUT-2 |

---

## 1. 文脈・問題

設計骨子（blueprint）の初版にはセクション 8「ブログ・お役立ち記事」が含まれていた。real045 のような本格サイトを参考にすると、継続発信は「事業者感」「専門家感」の演出に有効。

しかし、運用者は 1 名で、記事執筆・SEO 運用の継続体制は未確立。半端な状態でブログを置くと「更新が止まっているサイト」になり、印象を逆に下げる。

## 2. 検討した選択肢

| 案 | 概要 | 利点 | 欠点 |
|---|---|---|---|
| A. ブログを実装する | 設計骨子通り | 「専門家感」が出る | 記事 0 件のままだと逆効果、運用負荷 |
| B. **ブログを実装しない** | セクション削除 | 余計な期待を生まない、運用負荷ゼロ | "事業者感" を別手段で代替する必要 |
| C. 記事 5 本以上揃ったら実装 | 段階的 | 安全 | "Yes/No" の意思決定を先送り |

## 3. 決定

**案 B: 当面実装しない** を採用。将来 FUT-2（記事 5 本以上の供給体制ができたとき）に再考。

### 理由

1. 記事 0 件の「最新の記事」セクションは逆効果
2. 運用負荷ゼロを優先
3. 代替: Voice カード（受講生対談）、Achievement（銀の盾）、Profile ストーリーで "本物感" を担保

## 4. 結果

- 「ブログ」メニューを HTML から削除
- フッターからも削除
- 将来の起動条件: 記事 5 本以上の供給合意 + 月次更新の運用フロー確定

## 5. トレードオフ

- SEO の長尾キーワード流入を逃す（本サイトはブランディング目的で SEO 重視ではないため許容）
- 「ちゃんとした事業者感」は他の演出（Voice / Achievement / Profile）で代替
